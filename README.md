# finance-tracker-data 🔒

Repositorio **privado** de datos para la app FinanceTracker.

> ⚠️ Mantén este repositorio **siempre privado**.
> Los archivos aquí son generados y actualizados automáticamente por la app.

## Estructura

```
data/
├── users.json          ← Usuarios registrados (contraseñas en hash SHA-256)
└── transactions.json   ← Todas las transacciones de todos los usuarios
```
